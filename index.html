Mis Proyectos

Proyecto
Animalitos Veterinaria

Construcción y desarrollo de una página web enfocada en una veterinaria.

Este proyecto se concretó con las especificaciones dadas por el cliente en el lenguaje C#, el cual necesitaba una página con varios apartados útiles.

Se aportaron los diseños finales y también se programaron los apartados, como productos. Hubo una constante revisión y apoyo al trabajo de otros integrantes en front-end y back-end.

Proyecto
Hades

Se requería de un mantenimiento que ayudara a un programa antiguo de facturación a adaptarlo de un programa nativo de consola en C# a una interfaz gráfica con Windows Forms, para poder tener una mejor comprensión de cuándo está funcionando y poder detenerlo e iniciarlo a petición del usuario.

El principal aporte fue analizar para adaptarlo y crear las nuevas funcionalidad para ejecutarlo asi como las medidas contra errores e iniciar el programa con parametros del usuario.

Proyecto
Monitor de Calidad de Aire

Este proyecto se enfocó en tener la capacidad de monitorear el estado del aire para poder analizar la calidad mediante una app que se conectaba a una base de datos con actualización en tiempo real.

Mi aporte principal fue el desarrollo de la app para un dispositivo móvil y el diseño de la base de datos de manera simple.
